# project-metadata

Using an issue tracker for controlling GO project metadata and project discussion.

## Claude Code Support

This repository includes a [`CLAUDE.md`](CLAUDE.md) file that enables Claude (via [Claude Code](https://claude.ai/claude-code)) to assist with project management operations — status reports, project creation, lifecycle management, and audits. It documents the structure of this repo, how issues and org project boards are linked, and the `gh` CLI commands needed for all operations. It is intended for use by both human users and Claude.
